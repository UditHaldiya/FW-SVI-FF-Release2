Command 129, 40 - Read Conversion Coefficients

Command Type: Read
Allowed modes: Fail Safe, Manual, Setup, Normal
Description: Read parameters for converting from external representation (floating point) to internal (integer) and back.
Let x be internal number and X external. The conversion formulas are as follows:
X = (i2f_slope*(x - Xintercept)*2**(-i2f_shift) + Yintercept)/(10**order)
y = round(X*10**order)
If y does not fit 32-bit integer, reject
x = round(f2i_slope*(y-Yintercept)*2**(-f2i_shift) + Xintercept)
